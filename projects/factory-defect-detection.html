Factory Defect Detection

Automated visual inspection system for manufacturing quality control with 99.2% accuracy — reducing defect escape rate by 85%.

ClientPrecision Manufacturing Co.
Duration5 Months
RoleLead AI Engineer

The Problem

A precision manufacturing facility was experiencing a 3.2% defect escape rate with manual visual inspection — costing approximately $1.2M annually in returns and rework. Human inspectors could only sample 15% of products due to production line speed constraints, and fatigue led to inconsistent quality judgments.

The Solution

I developed an automated visual inspection system using YOLOv8 for defect detection and classification, deployed as a Docker-based microservice integrated directly into the production line. The system inspects 100% of products at line speed (120 units/minute) and classifies defects into 12 categories including scratches, dents, misalignments, and surface contamination.

Key Challenges

  • High-Speed Inspection: Engineered a pipeline that processes 120 units/minute with <100ms inference latency per unit using ONNX-optimized models.
  • Defect Class Imbalance: Addressed severe class imbalance (some defects occur in <0.1% of units) using focal loss and synthetic data augmentation.
  • Production Integration: Designed a non-blocking inspection microservice that interfaces with PLC controllers for automatic rejection of defective units.
  • Model Drift Monitoring: Implemented automated drift detection with weekly model retraining triggers to maintain accuracy as product lines evolve.
