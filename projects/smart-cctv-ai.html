Smart CCTV AI

Real-time anomaly detection and behavior analysis for security surveillance using edge-optimized deep learning on NVIDIA Jetson Orin.

ClientSecurity Solutions Provider
Duration4 Months
RoleLead AI Engineer

The Problem

A security solutions provider needed an intelligent surveillance system that could automatically detect anomalies — intrusions, loitering, unattended objects, and crowd formation — in real-time across hundreds of camera feeds. Traditional rule-based systems generated too many false positives, while cloud-based AI solutions introduced unacceptable latency and bandwidth costs.

The Solution

I designed and deployed an edge-native AI pipeline that runs YOLOv8-based object detection and custom behavior analysis models directly on NVIDIA Jetson Orin devices at each camera cluster. The system processes video streams locally at 30 FPS per camera, only sending alert metadata to the central monitoring dashboard — reducing bandwidth by 95% compared to cloud-based alternatives.

Key Challenges

  • Real-time Performance: Achieving 30 FPS inference on edge hardware required aggressive model optimization with TensorRT, INT8 quantization, and custom CUDA kernels.
  • False Positive Reduction: Implemented a multi-frame temporal filtering pipeline that reduced false positives by 92% while maintaining detection sensitivity.
  • Multi-Camera Synchronization: Developed a time-synchronized frame buffering system to handle up to 16 camera streams per Jetson device.
  • Low-Light Performance: Fine-tuned models on a custom dataset with night-time augmentation to maintain >95% accuracy in challenging lighting conditions.
